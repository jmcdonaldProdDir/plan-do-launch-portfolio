ShelfMonitor for Advisors Configuration

ShelfMonitor for Advisors can be configured to meet the requirements of the dealer. The configuration options include:

SMfA CONFIGURATIONS

As an ICOM Admin, go to the ConfigAdmin of the CompliancePlatform to configure SMfA.

  • CONFIG ADMIN - SETTINGS
  • Turning on/off SMfA 
  • Turn on/off the ability for Advisors to set remove/add data points they monitor
  • Setup email address and names for the alert sender, and alert reply
  • CONFIG ADMIN - MANAGE SMFA DATAPOINTS
  • Set the data points to monitor by security type. SMfA data points setup by security type (recommend three to four key datapoints; e.g. CIFSC Category, MER, Prospectus Risk Rating, 3 Year Compound Return
  • CONFIG ADMIN - CONTACTS
  • Set the contacts for position file ingestion alert. Each time a positions file is ingested these contacts will receive an email alert with the results of the ingestion process.
POSITIONS INGESTION

Ideally, the client provides InvestorCOM with a JSON file through SFTP into their folder in our Azure platform. When the client follows this procedure, they can upload a positions file on a routine basis through automation with out InvestorCOM ADMIN assistance. 

We currently support two types of SMfA position ingestion:

  • JSON automation
  • Manual CSV

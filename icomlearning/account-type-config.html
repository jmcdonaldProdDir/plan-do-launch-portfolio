Account Type ConfigurationsDOWNLOAD EXCEL TEMPLATE

The RolloverAnalyzer Cost Model has the following features:

  • Add multiple IRA account types.
  • User-defined labels for each Account Type definition 
  • Plan amount is added manually 
  • Account and Investment fees are percentages 
    • Cost 1 (account fees) is mandatory 
    • Cost 2 (investment fees) is optional. If not configured, it will not display on the form. 
  • Total Costs sum Account and Investment fees 
  • There are three Other Costs fields 
    • User-defined labels 
    • Declared as either percentages or dollars 
    • Other costs not defined, will not display 
  • All the information is displayed in the Recommendation Analysis 

Your Configurations

Follow these steps to configure the Account Types:

  • Revise and/or approve the two default account types - Advisory and Brokerage
  • Add user defined Account Types

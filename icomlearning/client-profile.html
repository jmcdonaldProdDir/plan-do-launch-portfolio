Protected Fields

The following six fields are protected and required when processing a rollover. These fields cannot be changed.

First Name Client Last Name
Client Email  Birth Date (reformatted)
Recommendation Type Decision Support
- Recommendation
- Marketing
- Education

Configurable Fields

The following four fields are configurable and not-required when processing a rollover. These fields can be changed. New fields and drop-downs may be added.

Investment Objective
- Growth
- Income
- Growth and Income

- Change of Employment

- In-Service Distribution
Time Horizon

- Medium Term (5-10 Years)

-  Moderate
-  Moderately Conservative
